Para mejorar el rendimiento de los modelos, no va tanto por los modelos en si, sino que se debe buscar la forma de aumentar el dataset. Existen muchas técnicas para lograr esto. En particular, para imágenes se puede a partir de un ejemplo, realizar tratamiento de imágenes para aumentar la cantidad de datos, como aumentar el contraste, realizar recortes, invertir el sentido, etc.
