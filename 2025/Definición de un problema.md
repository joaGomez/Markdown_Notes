Para poder definir un problema, es necesario un sujeto, un grupo o comunidad, que identifique una falta o una necesidad. Debe haber una situación la cual no es ideal, y se desearía que exista un proceso o un sistema que mejore esto.
### Componentes de un problema
- Metas: Objetivo que se desea alcanzar.
- Datos: Elementos para analizar una situación problemática.
- Restricciones: Límites que existen para llevar a cabo la meta.
- Operaciones: Procedimientos para resolver el problema.
### Criterios para la clasificación de problemas
- **Tipo y diversidad de variables intervinientes:**
	- *Problemas cerrados:* Se conocen las variables que componen el problema, y existe una única solución a este.
	- *Problemas abiertos:* No todas las variables están definidas, y existen múltiples soluciones.
- **Métodos predominantes  para su resolución:**
	- *Problemas lógicos:* Se obtiene un resultado a partir de ciertas premisas, aplicando diferentes procedimientos algorítmicos.
	- *Problemas teóricos:* La solución que brinda es una explicación de un suceso o situación dada. Se manifiesta como una teoría.
	- *Problemas prácticos o técnicos:* Se utilizan procedimientos algorítmicos, pero su finalidad es de una acción concreta.
### Estrategias de resolución de problemas
Procedimientos que se aplican de modo controlado, dentro de un plan diseñado deliberadamente con el fin de conseguir una meta fijada.

<u>Preguntas fundamentales de este proceso:</u>
- ¿Qué sé y qué desconozco?
- ¿Qué quiero obtener?
- ¿Qué variables lo componen?
- ¿Existen situaciones análogas?
- ¿Puede dibujarse un esquema, figura o gráfico que facilite la comprensión del problema?
